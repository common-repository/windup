== Description ==

Cut your posts automaticaly after x number of words, or letters. U can stil use <!--more--> tag
and manual excerpt. If non of them are used, plugin will automaticaly cut post at site other that single post site. Make sure taht in your main loop post are displayed by the_content, there is no need to check is_home, puligin do it by itself.


== Installation ==


1. Instal by upload .zip file
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set the initial values in plugin settings, and save them.
